## Second Brain — Portable AI Maps

This folder is the portable layer between the vault and any AI tool.
Files here are plain markdown — they travel to Claude, Gemini, Cursor, or any tool that reads text.

**Philosophy:** File over AI. Knowledge belongs to the vault, not the tool.
